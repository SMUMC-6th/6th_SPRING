# 6th_SPRING
6th SMUMC 스터디 SPRING

### 📋 Commit Message Convention
| Tag | Description |
| :---: | --- |
| `Feat` | 새로운 기능 추가 |
| `Fix` | 버그 수정 |
| `Docs` | 문서 추가, 수정, 삭제 |
| `Test` | 테스트 코드 추가, 수정, 삭제 |
| `Style` | 코드 형식 변경 |
| `Refactor` | 코드 리팩토링 |
| `Perf` | 성능 개선 |
| `Build` | 빌드 관련 변경사항 |
| `Ci` | CI 관련 설정 수정 |
| `Chore` | 기타 변경사항 |
